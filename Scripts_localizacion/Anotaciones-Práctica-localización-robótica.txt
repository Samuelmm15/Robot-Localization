Práctica de localización

La localización de un robot se aplica cuando el robot ideal se ha deviado del real, por tanto, cuando se detecta que el robot ideal se ha desviado del real se aplica la búsqueda, esta se encarga de analizar cada uno de los píxeles de la imagen para ver si se encuentra el robot real, una vez se encuentra el robot real se envia a la posición. Para que el robot pueda funcionar de manera correcta, se debe de tener al menos dos balizas ya que si se tiene una única baliza no se puede calcular la posición del robot real y por tanto el robot lo más seguro es que se pierda.
