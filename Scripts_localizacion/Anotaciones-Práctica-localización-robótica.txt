Práctica localización robótica

La práctica de localización, la acción u operación que se debe de realizar con los script, se basan en que cuando el robot se desvía de la ruta debido a los errores, se debe de aplicar la localización en ese momento, para este caso, la localización que debe de ser implantada es la localización mediante balizas. Estas balizas, sirven para poder determinar si el robot se devía mucho de estas balizas, se debe de aplicar el algoritmo de localización para que el robot se acerque a las distintas balizas que existen dentro de esta ruta.
